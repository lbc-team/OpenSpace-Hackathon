# 作品名称: RenToken

## 作品简介

RenToken是一个面向全世界的房租RWA项目。 致力于解决房租收益获取流动性难的问题。我们的创新点是：

1. 链下房租收益打包估值质押，收益权和产权分离
2. 链上注册房产metadata合约 和 有收益池属性，支持发行全流程的erc20代币
3. KYC投资人直接购买代币
4. 运行期分红直接用usdc分配到收益池，代币持有者可以实时提取
5. 提供支持KYC的AMM流动性池，支持链上流动性
6. RBAC权限管理模型支撑
